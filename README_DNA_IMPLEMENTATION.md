## 📊 Technical Specifications

### DNA Analysis Metrics
- **GC Content**: Percentage of Guanine + Cytosine bases
- **AT Content**: Percentage of Adenine + Thymine bases
- **Nucleotide Counts**: Individual A, T, G, C counts
- **Genetic Distance**: Deviation from typical spider DNA (46% GC)
- **Confidence Score**: Based on GC content matching
- **Species Matches**: Top 5 matches with color-coded status

### Supported Formats
- `.fasta` - Standard FASTA format
- `.fa` - Short form FASTA
- `.txt` - Text format with FASTA content

### Performance
- Parsing speed: < 100ms
- Sequence size: Optimized for < 10,000 bp
- Tab switching: Instant
- Memory efficient: No memory leaks
